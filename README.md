The project is structured into five essential files:

| File | Purpose | Notes |
| :--- | :--- | :--- |

| **`traits_and_fusing.html`** | **Secondary Content Page** | Details all trait multipliers (e.g., 10x), Lucky Block costs, and essential fuse recipes. |
